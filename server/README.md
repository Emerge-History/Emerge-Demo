# emerge-demo-server

## Usage

### Change Environment
open ./src/config.js and change the 'env' field to 'development, test, production' 

```js
// for example
env:'test' => env:'development'
```

### start the server 
```js
npm start
```

### use babel transform es6 => es5
```js
npm run build 
```

### run test
```js
npm run test
```

### run the build server
```js
npm run serve
```
