# emerge-demo-server

## Usage

### select NODE_ENV
NODE_ENV can be test, dev, pro  default:dev

for example
```js
set NODE_ENV=test & npm start
```

### start the server 
```js
npm start
```

### use babel transform es6 => es5
```js
npm run build 
```

### run test
```js
npm run test
```

### run the build server
```js
npm run serve
```
